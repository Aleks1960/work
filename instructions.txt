git <command> 

Tab - автодополнение (если других вариантов нет)
Tab x 2 - показать возможные варианты автодополнения

git status - показать статус репозитория

git add <file> - добавить файл file
git add . - добавить все файлы которые после git status shown as red


Типичный способ работы с гит:

git add <files>
git commit -m "message"
git push origin master - залить всё в ветку мастер

Скачать изменения сделанные другим разработчиком
git pull
